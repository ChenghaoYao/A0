1. 能量收入函数 E_in(r)

```
E_in(r) = η * A * L_sun / (4πr²)
```

· r: 到太阳的距离
· η: 系统综合效率（光热转换、传输损耗等）
· A: 能量收集面积
· L_sun: 太阳总光度 (~3.86×10²⁶ W)

2. 能量支出函数 E_out(r)

```
E_out(r) = E_thermal(r) + E_propulsion + E_life_support
```

· E_thermal(r): 热管理能耗 - 这是距离r的强函数，近太阳时急剧上升
· E_propulsion: 维持轨道和机动的能耗
· E_life_support: 系统自身维持的能耗

```
E_in(r_profit) = E_out(r_profit)
```
