一、为什么必须依赖体素边长是 0.5？

你的可逆公式是：

```
lat = originLat + z * (meterPerVoxel / 111320)
lng = originLng + x * (meterPerVoxel / (111320 * cos(originLat)))
alt = originAlt + y * meterPerVoxel
```

逆向公式是反过来：

```
z = round((lat - originLat) / (meterPerVoxel / 111320))
x = round((lng - originLng) / (meterPerVoxel / (111320 * cos(originLat))))
y = round((alt - originAlt) / meterPerVoxel)
```

只有当 meterPerVoxel / 111320 能写成有限小数，且经纬度步长不是无理数时，round 才能完美回到原整数。

你选 meterPerVoxel = 0.5，是因为：

· 0.5 / 111320 是一个有理数
· 经纬度步长可逆
· 体素边长 0.5 米 → 整数坐标 (x, y, z) 能唯一映射回 GPS，且再逆回来不丢坐标

如果你选 0.3 或 0.7，除出来是无理数，经纬度步长无限小数，round 就可能跳格。
体素边长 0.5 不是你“偏好”，是你体系可逆的数学支点。

二、怎么计算“均匀分布奇异值”？

对于一个点序列 P0, P1, P2, ..., Pn：

1. 计算相邻点之间的欧氏距离
      d_i = |P_i - P_{i-1}|
2. 计算这些距离的标准差（或方差）
3. 标准差越小 → 点分布越均匀 → 奇异值越小
      标准差越大 → 点分布越不均匀 → 奇异值越大

你也可以用“最大间距 / 最小间距”作为奇异值。

· 比值接近 1 → 均匀
· 比值远大于 1 → 不均匀

你甚至可以定义自己的公式：

奇异值 = (最大间距 - 最小间距) / 平均间距

你是定义者，你说了算。

1. 判断“点数量是否够用”

· 奇异值小 → 点分布均匀 → 路径平滑 → 精度够
· 奇异值大 → 点分布不均 → 某段太疏 → 需要加密

如果奇异值 > 0.1，自动在稀疏段插点。

二、JED 的数学定义（你说了算）

      d_i = |P_i - P_{i-1}|，其中 i = 1..n

3. JED 可以定义为：

          JED = σ_d / μ_d

   · 极差版：
          JED = (max(d) - min(d)) / μ_d

JED 越小 → 点分布越均匀 → 路径越平滑
JED 越大 → 点分布越不均匀 → 可能存在稀疏段或冗余密集段

JED（均匀分布奇异值）

全称：均匀分布奇异值
缩写：JED
拼音拆解：Junyun（均匀）+ Eccentric（奇异）+ Distribution（分布）

定义
衡量离散点序列（QD/D2/ZD）中，相邻点之间距离的均匀程度。
JED 越小 → 点分布越均匀（路径越平滑）
JED 越大 → 点分布越不均匀（存在稀疏或过密段）

计算公式
给定点序列 P0, P1, ..., Pn：

1. 计算相邻点之间的距离
      d1 = |P1 - P0|
      d2 = |P2 - P1|
      ...
      dn = |Pn - P(n-1)|
2. 计算这些距离的平均值 μ 和标准差 σ
3. JED = σ ÷ μ
      （这个比值没有单位，只反映均匀程度）

替代公式（可选）
JED = (最大距离 - 最小距离) ÷ 平均距离

使用场景

· 路径生成后自动检查质量
· 判断哪里需要加密或抽稀
· 作为导出前的验收标准

---

JDM（均匀分布点序列密度）

全称：均匀分布点序列密度
缩写：JDM
拼音拆解：Junyun（均匀）+ Distribution（分布）+ Midu（密度）

定义
你期望的相邻点之间的目标距离。单位：毫米。
JDM 是输入参数，用来告诉 D3“点应该隔多远放一个”。

· JDM = 0.5 → 每 0.5 毫米一个点
· JDM = 0.1 → 每 0.1 毫米一个点

使用方式

1. 直线路径
      已知起点 QD、终点 ZD、总长度 L
      期望点数 N = L ÷ JDM
      在起点和终点之间均匀插入 N-2 个点
2. 曲线路径（如圆）
      已知周长 C
      期望点数 N = C ÷ JDM
      按角度均匀分布 N 个点
3. 已有路径的优化
      计算实际相邻点间距
      如果某段间距 > JDM × 1.5，则加密（插入更多点）
      如果某段间距 < JDM × 0.5，则抽稀（删掉中间点）

与 JED 的关系

· JDM 是你设定的目标（指令）
· JED 是 D3 计算出来的结果（反馈）
· 闭环流程：你调 JDM → D3 生成路径 → 计算 JED → 你判断是否满意 → 不满意就再调 JDM
